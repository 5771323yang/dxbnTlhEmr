# Python图书管理系统 python1270

## 项目概述

本项目是一款基于Python的图书管理系统，具备图书信息管理、借阅者信息管理以及借阅记录管理等功能，为图书管理提供了全方位的解决方案。

## 技术栈

- 后端框架：Flask
- 数据库：MySQL
- 开发工具：Navicat，Pycharm

## 功能模块

- 图书信息管理
  - 图书录入
  - 图书查询
  - 图书修改
  - 图书删除
- 借阅者信息管理
  - 借阅者信息录入
  - 借阅者信息查询
  - 借阅者信息修改
  - 借阅者信息删除
- 借阅记录管理
  - 借阅记录查询
  - 归还记录管理

## 系统角色

- 系统管理员：负责图书和借阅者信息的管理
- 借阅者：可以进行图书借阅和归还操作

## 运行环境

- Python
- MySQL

## 部署步骤

1. 准备Python和MySQL环境
2. 安装Flask框架和MySQL数据库
3. 导入数据库结构
4. 运行后端程序

## 目录结构

```
project/
│
├── app/                # Flask应用程序目录
│   ├── templates/      # HTML模板目录
│   └── static/         # 静态文件目录
│
├── venv/               # 虚拟环境目录
│
├── migrations/         # 数据库迁移文件目录
│
└── run.py              # 程序启动文件
```

## 核心亮点

- 使用轻量级Flask框架，系统运行效率高
- 结合MySQL数据库，数据存储稳定可靠
- 搭建过程简单明了，适合学习和项目实践
- 功能全面，满足图书管理的各种需求
